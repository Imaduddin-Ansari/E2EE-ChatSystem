# MERN Application

## Running the Application

Follow these steps to run the MERN application locally:

### 1. Clone the repository
git clone https://github.com/your-username/your-repo.git
### 2. Open the project in your IDE
Open the cloned project folder in your preferred IDE (e.g., VS Code).
### 3. Install dependencies
Backend:
cd backend
npm install
Frontend:
cd ../frontend
npm install
### 4. Ensure MongoDB is running
Make sure MongoDB Compass or your MongoDB server is running before starting the app.
### 5. Start the application
Backend:
cd backend
npm start
Frontend:
cd frontend
npm start
### 6. Access the application
Backend API default URL: http://localhost:3001
Frontend React app default URL: http://localhost:3000
Your MERN application should now be fully running locally
